# Theme

- [dark-theme-editor](https://themes.gohugo.io/themes/dark-theme-editor/)

### Install theme

```bash
git submodule add -f https://github.com/JingWangTW/dark-theme-editor.git themes/dark-theme-editor
```

### Re-init theme

```bash
git submodule update --init --recursive
```

# Start server

```bash
hugo server -D
```
